📦 Installation

1

Install a userscript manager

Get Tampermonkey (Chrome/Edge) or Violentmonkey (Firefox).

2

Install the script

Click the "Install Userscript" button above or visit the Greasy Fork page and click "Install".

3

Reload IXL

Go to any IXL page – the mods will activate automatically. Open your profile to customize images.

ActionKey / Method
Open profile customizationClick your name in the top‑right → "Profile, Settings, & Customization"
Add profile picture from URLPaste URL in the input field and press Enter
Upload local imageUse the file picker next to the URL field
Delete a saved imageIn the gallery, click the ✖ button on the image
Toggle dark modeIn the profile panel, check "Dark Mode"
Change help button sideIn the profile panel, select left or right from the dropdown
Open lesson helpClick the "Help" button on any lesson page
